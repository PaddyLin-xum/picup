# 项目简介
基于[picee](https://github.com/jrainlau/picee)插件制作的项目picup，本项目是为了更好的上传图片到Github仓库，以得到依靠jsDeliver加速的图片链接（即Github图床）。现在目前优化是改变原项目生成图片链接。后期将优化界面显示，以及部分功能。更新可能较慢
